Property tools

One dataset, three views. Facts live in data.json; true cost, transfer tax, collateral and scoring all derive from it via lib.js — so the tools can't disagree about the same house.

Shortlist

All candidates scored and sortable. True cost derived; off-anchor properties gate-flagged.
02

Scorer

Score a new listing against the rubric, with the school veto. Emits a data.json snippet to paste in.
03

Financing
